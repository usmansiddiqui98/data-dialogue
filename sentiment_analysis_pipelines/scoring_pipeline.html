Scoring Pipeline¶¶

src.models.sentiment_analysis.scoring_pipeline.run_scoring_pipeline(input_df)¶

Execute the sentiment analysis scoring pipeline on the input DataFrame.

The pipeline consists of the following steps: 1. Preprocessing the input data. 2. Feature engineering. 3. Model selection and loading. 4. Prediction using the best model. 5. Saving the predicted sentiment and sentiment probabilities in an output DataFrame.

Parameters:

input_df (pandas.DataFrame) – The input DataFrame containing the raw reviews data.

Returns:

The output DataFrame containing the predicted sentiment and sentiment probabilities for each review.

Return type:

output (pandas.DataFrame)
